## 1. File and Workbook Management

| Shortcut | Functionality | Usage |
|--------|---------------|-------|
| Ctrl + N | Create new workbook | Start a fresh Excel file |
| Ctrl + O | Open workbook | Access existing files |
| Ctrl + S | Save workbook | Store current changes |
| F12 | Save As | Save with new name or location |
| Ctrl + P | Print workbook | Print sheets or entire file |
| Ctrl + W | Close workbook | Close current file |
| Alt + F4 | Exit Excel | Close Excel application |
