## Practice Questions Based on Sales Details Table

### 1. Title Formatting
- Create a table title named  
  **SALES DETAILS OF CUSTOMERS WHO HAS ORDER THE PRODUCTS**
- Place the title in the first row.
- Merge the title across all table columns.
- Set font size to 18.
- Apply bold style.
- Change font color to dark blue.
- Apply yellow fill color.
- Align the title to the center.

### 2. Header Structure
- Create three main headers:
  - PRODUCTS
  - CUSTOMERS
  - ORDERS
- Merge columns for headers only.
- Do not merge any data cells.

### 3. Column Mapping
- Under PRODUCTS:
  - PRODUCT_ID
  - PRODUCT_NAME
  - PRICE
  - QTY
- Under CUSTOMERS:
  - CUSTOMER_ID
  - C_NAME
  - LOC
  - MOBILENO
- Under ORDERS:
  - ORDER_ID
  - SALES
  - DATE
  - TYPE

### 4. Header Formatting
- Set header font size to 12.
- Apply bold style to all headers.
- Use light gray fill color.
- Align header text to center.

### 5. Data Formatting
- Use Calibri font for all data cells.
- Set font size to 11.
- Align text columns to the left.
- Align numeric columns to the right.
- Format MOBILENO column as Text.

### 6. Borders
- Apply thick border to the outer table.
- Apply thin borders inside the table.

### 7. Layout Adjustments
- Adjust column width to show full header names.
- Increase row height for the title row.
- Keep standard row height for data rows.

### 8. Freeze Panes
- Freeze rows so the title and headers remain visible while scrolling.

### 9. Restrictions
- Do not merge data cells.
- Use merge and unmerge only for headers.
- Do not apply fill color to data rows.
