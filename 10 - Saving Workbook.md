# Module 10 – Saving Files & Formats in Excel

## Why Saving Matters
Saving in Excel is not only storage.
It impacts version control, security, file size, and compatibility.
Choosing the wrong format can cause data loss or broken automation.

---

## 1. Where to Save Files

### Local Storage
- Saved on PC or hard drive
- Suitable for sensitive or offline data
- Fast access
- No automatic backup or version history

### Cloud Storage (OneDrive / SharePoint)
- AutoSave enabled
- Saves changes every few seconds
- Version history available
- Supports real-time collaboration

---

## 2. Essential Excel File Formats

| Format Name | Extension | Purpose |
|------------|----------|---------|
| Excel Workbook | .xlsx | Standard format, no macros |
| Macro-Enabled Workbook | .xlsm | Required for VBA macros |
| Binary Workbook | .xlsb | Large datasets, faster loading |
| PDF | .pdf | Final reports, non-editable |

---

## 3. Advanced Saving Options

### Password Protection
- Password to Open blocks file access
- Password to Modify allows view-only access

### CSV Format
- Used for SQL, Python, and web uploads
- Saves only values
- Removes formulas, formatting, and multiple sheets

---

## 4. Legacy File Format

### Excel 97–2003
- Extension: .xls
- Used only for very old Excel versions
- Limited to 65,536 rows
- Avoid unless required

## Student Tasks – Format Decision Practice

### Task 1: Macro Safety
You created a report using VBA macros.
After reopening the file, macros are missing.

- Identify the mistake.
- Write the correct file format to prevent this issue.

---

### Task 2: Large File Optimization
An Excel file is 250MB.
It takes too long to open.
All formulas and data must remain.

- Which file format should be used?
- Why is this format suitable?

---

### Task 3: Management Report
You are sharing the final budget with senior management.
They must see totals but should not edit formulas.

- Which format should you choose?
- Explain the reason.

---

### Task 4: Database Transfer
You need to import Excel data into a SQL database.

- Which format will you save the file in?
- List two limitations of this format.

---

### Task 5: Storage Decision

- Where should the file be saved?
- Mention two advantages of this location.
