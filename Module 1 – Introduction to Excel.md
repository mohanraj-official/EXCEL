## 1. What is Excel

* Spreadsheet software developed by Microsoft
* Stores and manages data in tables
* Performs calculations using formulas and functions
* Creates charts and reports
* Automates work using macros and VBA

## 2. Key Uses of Excel

* Data entry and storage
* Calculations and analysis
* Data visualization using charts and pivot tables
* Task automation
* Reporting and tracking

## 3. Brief History of Excel

* Released in 1985 for Apple
* Windows version released in 1987
* Introduced charts, VBA, and automation over time
* Now part of Microsoft 365 

## 4. Why Excel Is Important

* Manages large datasets
* Reduces manual calculation effort
* Converts data into visual insights
* Supports collaboration and sharing
* Available on Windows, Mac, web, and mobile
* Integrates with Word, PowerPoint, and Outlook

## 5. Why Data Analysts Must Learn Excel

* Cleans raw data using built-in tools
* Explores data using filters and pivot tables
* Performs basic statistical analysis
* Builds charts and dashboards
* Integrates with SQL, Python, and Power BI
* Required skill for most data analyst roles

## 6. Training Objective

* Understand Excel interface and basics
* Use formulas and functions confidently
* Clean and organize data
* Create charts and pivot tables
* Build simple dashboards
* Automate repeated tasks
* Apply Excel to real-world datasets
